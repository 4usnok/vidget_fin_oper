# Цель проекта

Разработать проект «Виджет финансовых операций» и нарастить его функциональность.

# инструкции по установке и использованию разработанных функций

1. Клонируйте репозиторий:
```
git clone git@github.com:4usnok/proj_1.git
```

2. Установите зависимости:
```
pip install -r requirements.txt
```

3. Использование разработанных функций:
* Функция filter_by_state принимает список словарей и опционально значение для ключа 
state (по умолчанию 'EXECUTED'). Функция возвращает новый список словарей, содержащий
только те словари, у которых ключ state соответствует указанному значению.
* Функция sort_by_date принимает список словарей и необязательный параметр, задающий
порядок сортировки (по умолчанию — убывание). Функция должна возвращать новый список,
отсортированный по дате (date).

# Инструкция по тестированию программы
1. Зайдите в пакет tests
2. Откройте модуль test_func.py
3. Для тестирования представлены функции, которые выполняют тот или иной функционал. Принцип их работы, описан выше.
4. Каждая функция, помимо своего параметра, также имеет специальную инструкцию assert,  которая принимает на вход выражение, значением которого должно быть 
True, иначе выбрасывается исключение.
5. Как только ввели свои данные, введите в консоли ключевое слово pytest
6. В результате, сможем увидеть: какое количество функций работает корректно.

# Принцип работы модуля scv_excel_work.py
1. Функции реализовывают считывание финансовых операций из CSV- и XLSX-файлов.
2. На вход подаётся Абсолютный путь до файла, в котором содержатся словари транзакций, основная роль функций -
выдать список словарей с транзакциями
3. функция `excel_trans` и `csv_trans` считывать данные файлов с расширением XLSX и CSV

Принцип работы модуля re_func.py
1. Функция `filter_transactions_by_description` - принимает список словарей с данными о банковских операциях 
и строку поиска,
    а возвращает список словарей, у которых в описании есть данная строка
2. Функций `count_operations_by_category` - принимает список словарей с данными о банковских операциях и список категорий 
операций,
    а возвращает словарь, в котором ключи — это названия категорий,
    а значения — это количество операций в каждой категории
3. Функция `main.py` в модуле `main` - является управляющей и центральной. С помощью этой функции, пользователь сможет
получить информацию из трёх файлов разного типа: json, csv, xlsx. Сам принцип работы, состоит из фильтрации 
по категориям(сортировка по дате, по рублю, по возрастанию, по определенному слову 
в описании)
